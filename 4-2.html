4-2 可逆反應與平衡

平衡不是停止，而是來回一樣快

一、可逆反應的基本概念

定義

正反應與逆反應可以同時進行的反應

A + B ⇌ C + D
→

正反應

反應物消耗，產生生成物

←

逆反應

生成物反應，回到反應物

二、可逆反應實例：氯化亞鈷

藍色（乾燥）

CoCl₂

⇌

加水 → 變粉紅

加熱 → 變藍色

注意：並非所有反應皆可逆，例如「燃燒」屬於不可逆反應。

三、動態平衡 — 粒子模擬

左(反應物)：10 | 右(生成物)：0 | 反應進行中...
「動態」平衡表示微觀下正逆反應仍在持續進行，只是速率相等所以巨觀上粒子數量不再變化。像兩端人數相同的旋轉門。

平衡兩大關鍵：

  • 正速率 = 逆速率

  • 外觀不變，但微觀仍在反應

● 藍色 = 反應物（左側）

● 紅色 = 生成物（右側）

● 中間虛線 = 反應分界

● 平衡時兩邊粒子數穩定

四、R-t 圖判讀

剛開始：正反應最快，逆反應為 0

反應中：正變慢，逆變快

達平衡：兩速率交會，成水平線

五、平衡三條件

🔄

可逆反應

反應具備雙向進行的可能性

📦

密閉系統

物質不能隨意離開或進入

🌡️

環境穩定

溫度、壓力必須固定不變

六、勒沙特列原理

「當環境改變時，系統會朝向
抵消／減少改變的方向移動。」

增加某物 → 消耗它
減少某物 → 產生它

七、勒沙特列實戰模擬

改變濃度

改變溫度

反應物
生成物
⚖️ 系統平衡中

八、考前陷阱題預警

Q1

平衡時，正反應速率等於逆反應速率？

YES！

這是平衡唯一的黃金定義。

Q2

平衡時，反應物濃度一定等於生成物？

NO！

平衡只要濃度「不再變動」即可，兩邊不一定相等。

4-2 重點總整理

「平衡就是來回一樣快，
改變條件就會往抵抗改變的方向移動。」

速率相等
正反應 = 逆反應
外觀不變
顏色、濃度固定
仍在反應
微觀下的動態變化

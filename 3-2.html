3-2 酸和鹼

從解離觀點看酸鹼的化學本質

一、酸與鹼的解離定義

酸 (Acid)

溶於水可解離出 H⁺

HCl → H⁺ + Cl⁻

H₂SO₄ → 2H⁺ + SO₄²⁻

CH₃COOH → H⁺ + CH₃COO⁻

通式：H⁺ + 酸根離子

鹼 (Base)

溶於水可解離出 OH⁻

NaOH → Na⁺ + OH⁻

Ca(OH)₂ → Ca²⁺ + 2OH⁻

NH₃ + H₂O → NH₄⁺ + OH⁻

通式：金屬離子 + OH⁻

二、酸的共同性質

1

水溶液有酸味，且溶於水會放熱。

2

電解質：水溶液可導電。

3

與活性金屬反應 → 產生 H₂ (氫氣)。

4

與碳酸鹽反應 → 產生 CO₂ (二氧化碳)。

HCl + Zn (鋅片)

產生 H₂↑

HCl + CaCO₃

產生 CO₂↑

三、鹼的共同性質

1

有澀味、具滑膩感，溶於水亦會放熱。

2

電解質：水溶液可導電。

3

具清潔作用：可溶解油脂。

注意：鹼不與金屬、碳酸鹽反應。

NaOH 水溶液 + 油脂

油脂被鹼溶解（皂化）

四、酸鹼指示劑顏色變化

指示劑種類 酸性 中性 鹼性
石蕊試紙 變為 紅色 不變色 變為 藍色
酚酞指示劑 無色 無色 變為 紅色
廣用試紙 紅～黃 綠色 藍～紫

五、常見化合物特點

常見酸

  • ● 硫酸 (H₂SO₄)：強酸，稀釋須「酸加水」。
  • ● 鹽酸 (HCl)：有刺激性，用於除水垢。
  • ● 硝酸 (HNO₃)：強酸，具氧化性。
  • ● 醋酸 (CH₃COOH)：弱酸，食醋成分。

常見鹼

  • ● 氫氧化鈉 (NaOH)：強鹼，清潔劑主成份。
  • ● 氫氧化鈣 (Ca(OH)₂)：石灰水，檢測 CO₂。
  • ● 氨水 (NH₃)：弱鹼，具刺激臭味。
